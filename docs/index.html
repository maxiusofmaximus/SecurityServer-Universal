Instalación

Requisitos: Node.js ≥ 16.0.0, NPM ≥ 8.0.0

Opción 1: Desde GitHub (Recomendado)

# Clonar el repositorio
git clone https://github.com/tu-usuario/SecurityServer-Universal.git

# Navegar al directorio
cd SecurityServer-Universal

# Instalar dependencias
npm install

# Iniciar el servidor
npm start

Opción 2: Con NPX

# Instalación rápida
npx security-server-universal init

# O instalación global
npm install -g security-server-universal

# Iniciar
security-server start

Verificar Instalación

# Verificar que el servidor esté funcionando
curl http://localhost:3000/api/status

# O usar el CLI
node cli.js status

Ejemplos de Uso

Escanear Proyecto

Escanea tu proyecto en busca de vulnerabilidades:

# Escaneo completo
node cli.js scan ./mi-proyecto

# Escaneo profundo
node cli.js scan ./mi-proyecto --deep

# Solo auditoría de dependencias
node cli.js audit ./mi-proyecto
Crear Proyecto Nuevo

Crea un nuevo proyecto con seguridad integrada:

# Proyecto Node.js
node cli.js create mi-app --template nodejs

# Proyecto React con TypeScript
node cli.js create mi-react-app --template react

# Proyecto Python
node cli.js create mi-python-app --template python
Listar Plantillas

Ve todas las plantillas disponibles:

# Listar todas las plantillas
node cli.js templates

# Ver detalles de una plantilla
node cli.js templates --show nodejs
Configurar Seguridad

Aplica configuraciones de seguridad personalizadas:

# Configuración interactiva
node cli.js configure ./mi-proyecto

# Configuración específica
node cli.js configure ./mi-proyecto --level strict
